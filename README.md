# C++实现顺序表的基本操作
- 顺序表操作内容：
初始化、添加元素、删除元素、清空、销毁。

- 顺序表特点：
只有一个“第一个元素”和“最后一个元素”。除了头和尾，每个元素仅一个前驱，一个后驱。
